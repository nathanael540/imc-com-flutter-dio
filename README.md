# Calculadora de IMC com Flutter - DIO Santander Bootcamp 2023

Versão com packeage com Hive

### Checklist do Desafio
- Criar classe IMC (Peso / Altura)​
- Altura ler em tela de Configurações​
- Ler dados no app​
- Calcular IMC ​
- Gravar dados no Hive ou SQLite​
- Exibir em uma lista​
